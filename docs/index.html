What this is

This tool connects Grok-generated images and video to your Resolve project without needing Resolve Studio or the Higgsfield Workflow Integration plugin.

Typical flow:

  1. Generate or download Grok media.
  2. Files land in video/ or image/ inside your public folder.
  3. Import into the active bin you have selected in the Media Pool.
Works on Resolve Free using Utility scripts under Workspace → Scripts → Utility.

Requirements

  • macOS (tested; metadata scan uses macOS download tags)
  • DaVinci Resolve with Fusion scripting enabled
  • Python 3 on your Mac
  • xAI API key for chat and generation: export XAI_API_KEY=your-key
  • A clone of this repo on disk
The Higgsfield .pkg plugin is a separate Studio-only Workflow Integration. This project is the Resolve Free workaround: folder + scripts + terminal bridge.

Install

1. Clone the repo

git clone https://github.com/fornevercollective/grok-public-folder.git
cd grok-public-folder

2. Pick your folder path

Scripts default to /Users/tref/film/grok-public-folder. After cloning elsewhere, either clone to that path or update paths in:

  • resolve/utility/Grok Panel.py — GROK_ROOT
  • resolve/lua/grok_resolve.lua — ARTIFACTS
  • resolve/lua/grok_bridge.lua — BRIDGE

Python tools also respect GROK_PUBLIC_FOLDER if set before running bin/grok, bin/bridge, or bin/scan.

3. Install Resolve Utility scripts

chmod +x install-resolve.sh
./install-resolve.sh

Copies scripts into ~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility/. Restart Resolve or reload scripts if the menu does not update immediately.

4. Add the folder to Media Storage (optional)

In Resolve: DaVinci Resolve → Preferences → Media Storage, add your grok-public-folder path so you can browse video/ and image/ directly.

Grok Panel (recommended)

Open inside Resolve — no floating Terminal window required.

Workspace → Scripts → Utility → Grok Panel
ButtonAction
Scan Downloads Finds Grok downloads in ~/Downloads via macOS URL metadata, asks Yes/No, then moves into video/ or image/.
Import Imports all media from the public folder into the active Media Pool bin.
Preview Opens the latest files in Finder/Preview.
Refresh Updates the inventory list in the panel.
Select your target bin in the Media Pool before clicking Import. Thumbnails appear after import.

Terminal bridge

Lets the Resolve Lua or Python console talk to Grok for chat, image, and video generation. Resolve writes requests to bridge/request.json; Terminal writes responses to bridge/response.json.

Start the bridge (Terminal)

export XAI_API_KEY=your-key
/path/to/grok-public-folder/bin/bridge

Leave this running. It watches for requests from Resolve.

Terminal chat (no Resolve)

export XAI_API_KEY=your-key
/path/to/grok-public-folder/bin/grok

Commands inside the chat REPL:

/image mars sunset over dunes
/video slow orbit around a red planet --duration 10
/import
/help

Resolve console

Lua console

Load the bridge helper (use a function name — bare grok is invalid Lua):

dofile("/path/to/grok-public-folder/resolve/lua/grok_bridge.lua")
g()              -- help
ping()           -- test bridge
g("your prompt")
g("/image prompt")
g("/video prompt")
import()         -- import into active bin

Python console (Py3)

exec(open("/path/to/grok-public-folder/grok_load.py").read(), globals())
g("your prompt")
import_artifacts()

Downloads scan

Run from Terminal outside Resolve:

/path/to/grok-public-folder/bin/scan

Scans ~/Downloads for files whose macOS metadata references grok.com, x.ai, or related hosts. Shows a Yes/No dialog before moving files. Each moved file gets a .grok.json sidecar with source metadata.

Folder layout

grok-public-folder/
├── video/          clips land here

├── bridge/         lua/python ↔ terminal handoff

├── resolve/

│   └── utility/    panel + import scripts (copied by install-resolve.sh)
└── install-resolve.sh

Troubleshooting

ProblemFix
'=' expected near '<eof>' when typing grok in Lua grok alone is not valid Lua. Use g() or g("message").
Resolve() is nil in console Use Grok Panel instead, or run import from Utility scripts. Console context varies by Resolve version.
Imported 0 files video/ and image/ are empty. Generate or scan Downloads first.
Bridge times out Start bin/bridge in Terminal and set XAI_API_KEY.
Scripts missing in Resolve menu Run ./install-resolve.sh and restart Resolve.
Wrong folder in panel Update GROK_ROOT in resolve/utility/Grok Panel.py, then re-run install.
